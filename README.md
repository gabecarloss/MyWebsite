🌐 Personal Website

A modern personal website built to present my professional profile, projects, and contact information.
Focused on clean design, user experience, and clear presentation of technical skills.
----------------------------------------------------------------------------------------------------------------------------------
🚀 Overview:

This project is a responsive personal website developed using HTML and CSS.
It is designed to showcase my background, highlight selected projects, and provide an easy way to get in touch.

The goal is to create a professional online presence while applying solid front-end fundamentals and modern design principles.
----------------------------------------------------------------------------------------------------------------------------------
📌 Features: 
🧑‍💻 Personal introduction and professional summary
📚 About section with background and mindset
🛠️ Tech stack presentation
💼 Projects showcase with descriptions and links
📬 Contact section with email and social links
🎨 Modern UI with smooth animations and responsive layout
----------------------------------------------------------------------------------------------------------------------------------
🧠 Structure & Design:
-Clean and minimal layout with strong visual hierarchy
- Responsive design for desktop and mobile devices
- Smooth scroll and animation effects using JavaScript
- Organized sections: Home, About, Projects, Contact
- Emphasis on readability, spacing, and typography
----------------------------------------------------------------------------------------------------------------------------------
🛠️ Technologies Used:
HTML5
CSS3 (custom properties, responsive design, animations)
JavaScript (DOM manipulation and animations)
🧱 Project Structure
├── index.html      → Main structure of the website  
├── style.css       → Styling, layout, and animations  
└── img/            → Images and visual assets

📂 How It Works: 
Users can:

- Navigate through sections using the navbar
- Learn about my background and skills
- Explore selected projects with direct GitHub links
- Contact me via email or social platforms
